# Κρήτη - επόμενο βήμα μετά το Ηράκλειο

Ημερομηνία: 2026-05-23

## Τι αποφασίστηκε

Πρώτα έκλεισε η μοναδική εκκρεμότητα του Ηρακλείου και μετά ξεκίνησε το Ρέθυμνο.

Αυτό ήταν πιο ασφαλές από το να πάμε κατευθείαν σε νέο νομό, γιατί το Ηράκλειο είχε ακόμα ένα MEDIUM θέμα με δεύτερη `Ψιλή Άμμος`.

## Ηράκλειο

Προστέθηκε ξεχωριστή εγγραφή:

- Ψιλή Άμμος Καλών Λιμένων

Γιατί:

- Το OSM είχε δεύτερη `Ψιλή Άμμος` κοντά στα Στενά / Καλούς Λιμένες.
- Δεν ήταν σωστό να μετακινήσουμε την παλιά `Ψιλή Άμμος`, γιατί αυτή είναι άλλο σημείο.
- Μπήκε ως ξεχωριστή παραλία με χαμηλό confidence.

Τελικό audit Ηρακλείου:

- Παραλίες: 52
- BLOCKER: 0
- HIGH: 0
- MEDIUM: 0
- LOW: 5

Τα LOW είναι μόνο σημειώσεις ακρίβειας για merged sections/aliases.

## Ρέθυμνο

Πρώτα διορθώθηκε το scope του audit.

Το αρχικό audit Ρεθύμνου έφερνε λάθος παραλίες από Ηράκλειο και Χανιά. Προστέθηκαν γεωγραφικά bounds για `crete-crete-rethymno`, ώστε να μην ξαναγίνει αυτό.

Μετά αφαιρέθηκε από το Ρέθυμνο η `Καταλυκή`, γιατί οι συντεταγμένες της είναι στο Ηράκλειο. Η σωστή εγγραφή κρατήθηκε στο Ηράκλειο ως `Καταλυκή`.

## Τεχνική διόρθωση που έγινε

Ο generator κρατάει `metadata` μόνο όταν υπάρχει και `metadata.access.notes`.

Μερικές νέες εγγραφές είχαν `access.type` και `access.label`, αλλά όχι `access.notes`. Αυτό διορθώθηκε για τα νέα batches Ηρακλείου και Ρεθύμνου, ώστε η εφαρμογή να κρατάει σωστά:

- aliases
- terrain
- amenities
- access type
- water depth

## Checks που πέρασαν

```bash
npm run build:beach-data
npm run audit:beaches -- --island=heraklion --mode=deep --email-dry-run --refresh-external
npm run audit:beaches -- --island=rethymno --mode=deep --email-dry-run --refresh-external
npm run quality:beach-data
npm run content:audit
npm run lint
npm run build
git diff --check
```

Το `npm run build` πέρασε με το γνωστό Vite warning για μεγάλο chunk. Δεν σχετίζεται με τα beach data.

## Απόφαση

Η Κρήτη έχει τώρα καθαρό Ηράκλειο και καθαρό Ρέθυμνο σε blocker/high/medium επίπεδο.

Το επόμενο λογικό βήμα είναι να συνεχίσουμε με Χανιά, μετά Λασίθι.
